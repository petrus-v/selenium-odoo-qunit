===================
Selenium Odoo Qunit
===================

This is a CI launcher to run Odoo Qunit tests on multiple browsers at once using
selenium.

.. warning::

    This is not a lib to help you to write functional test in Odoo, it's intent
    is only to run Qunit tests page.
